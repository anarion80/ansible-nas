# Stashapp

Homepage: [https://stashapp.cc/](https://stashapp.cc/)

An organizer for your porn (and other media types)

## Usage

Set `stashapp_enabled: true` in your `inventories/<your_inventory>/group_vars/nas.yml` file.

The Stashapp web interface can be found at [http://ansible_nas_host_or_ip:9999](http://ansible_nas_host_or_ip:9999).
