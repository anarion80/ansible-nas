A self-hostable bookmark-everything app (links, notes and images) with AI-based automatic tagging and full text search

## Usage

Set `karakeep_enabled: true` in your `inventories/<your_inventory>/group_vars/nas.yml` file.

Karakeep web interface can be found at [http://ansible_nas_host_or_ip:3029](http://ansible_nas_host_or_ip:3029).
